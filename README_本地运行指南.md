# 📦 AI 提示词与图集管理器 • 本地客户端运行指南

欢迎使用 **AI 提示词与图集管理器** 的本地部署程序。
本软件的所有画图提示词、灵感库、多账号数据完全托管在您本地浏览器的 **离线安全沙盒** 内。

为了给您提供最丝滑的“一键客户端”双击启动体验，我们专门为您配置了 **一键自动化启动器（Launcher Suite）**。

---

## 🚀 极速一键启动步骤

### 1. 简易 Windows 一键启动
如果您使用的是 **Windows** 系统，只需：
1. **直接双击运行** 根目录下的 `双击本地启动(Windows).bat` 文件。
2. 启动器将进行极速自检，自动在您的浏览器中直连开启系统入口（`http://localhost:3000`）。

### 2. macOS 或 Linux 命令行启动
如果您使用的是 **macOS / Linux** 操作系统，只需：
1. 打开终端（Terminal）进入到当前项目根目录。
2. 第一次运行前请先为脚本赋予可执行权限：
   ```bash
   chmod +x 本地启动器(Unix-Mac).sh
   ```
3. 随后在终端双击或运行该启动器：
   ```bash
   ./本地启动器(Unix-Mac).sh
   ```

## 🛠️ 前置本地依赖（若您是首次运行）
要成功运行本系统，只要确保您的个人电脑上存在标准的 **Node.js** 环境即可：
* **官方推荐下载地址**: [Node.js 官方长期支持版 (LTS)](https://nodejs.org/zh-cn)
* 建议下载 **v18.x / v20.x / v22.x** 或更高版本。
* 启动器在被双击时会自动帮您检测，如缺少会给出清晰的提示指引。

## 🔑 安全退出与多账户登录
* **多账户沙盒**: 您的每一个本地注册账户对应的都是一个**独立的、互相完全隔离**的画集沙盒。
* **安全锁库**: 在页面右上角或左下角点击 **“安全退出”**，我们将向您展示特制的安全确认模态弹窗，确认后页面会退回本地登录界面，同时您的专属词库画集锁闭，任何不掌握暗号的本地旁人都无法窥探或读取。
